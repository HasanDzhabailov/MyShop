# MyShop

<h3>Tech stack</h3>
<ul>
<li>Kotlin based, Coroutines + LiveData</li>
<li>Koin for dependency injection</li>

<li>Jetpack: </li>
  <ul>
  <li>Lifecycle</li>
  <li>ViewModel</li>
  </ul>
<li>Architecture: MVVM with Clean Architecture, Multi module</li>
<li>Navigation Component</li>
<li>Retrofit2</li>
<li>okhttp3</li>
<li>Room</li>
<li>Gson</li>
<li>Glide</li>
</ul>
